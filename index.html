    Introduction

    Welcome to the Dreamscape Immersive API!

    This API documentation will provide explanations for our various endpoints with examples for the requests and responses you can excpect to see.

    Authentication

    Sample Request:

    curl --location --request GET 'https://api.hq.dev.techscapevr.com/common/sites' --header 'x-api-key: <Your API Key>'

    import requests

url = 'https://api.hq.dev.techscapevr.com/common/sites'
headers = {'x-api-key': '<Your API Key>'}

r = requests.get(url, headers=headers)
print(r.json())

    fetch("api.hq.dev.techscapevr.com/common/sites", {
    method: 'GET',
    headers: {
        'x-api-key' : "<Your API Key>"
    }
})
.then(response => response.json())
.then(data => console.log(data))

    Dreamscape uses API keys for authentication.

    Do not share your API key publicaly, this includes places like GitHub and in your own code. To make use of your API key include it as a header in your http request

    x-api-keys: <Your API key>

    To obtain a valid public API key for testing, please contact tclifton@dreamscapevr.com

    Errors

    HTTP Codes

    Dreamscape makes use of standard HTTP codes to show the status of a request. All responses will have an accompanying code that indicates success or failure.

    Error reponses will also have a message that describes the reason for the failure.

    Sample Error Message:

    {
    "message" : "Missing Authentication Token"
}
    Code Description
    200 - OK Everything is working as intended
    400 - Bad Request The request could not be completed
    401 - Unauthorized Your API key doesn't have permission to perform the request
    403 - Forbidden Missing API key
    404 - Not Found Request does not exist
    500 - Internal Server Error Server side error, an error on our side

    CRMLink Common API

    Purpose

    CRMLink Common API contains common functionality that will be required in multiple workflows. Information on Dreamscape retail sites, titles, etc can be accessed through this endpoint.

    Languages

    This endpoint has information on the languages that are availible for a specific Dreamscape title. Use title identifiers to search for titles.

    Request Information

    Sample Request:

    curl --location --request GET 'https://api.hq.dev.techscapevr.com/common/languages/alienzoo01' --header 'x-api-key: <public-api-key>'

    import requests

url = 'https://api.hq.dev.techscapevr.com/common/languages/alienzoo01'
headers = {'x-api-key': '{API Key}'}

r = requests.get(url, headers=headers)
print(r.json())

    fetch("https://api.hq.dev.techscapevr.com/common/languages/alienzoo01", {
    method: 'GET',
    headers: {
        'x-api-key' : "<Your API Key>"
    }
})
.then(response => response.json())
.then(data => console.log(data))
    Http request URL
    GET api.techscapevr.com/common/languages/{titleID}

    Request Parameters

    Sample Response:

    [
    {
        "language_code": "en",
        "language_name": "English",
        "written": true,
        "audio": true,
        "native_name": null
    },
    {
        ...
    },
    {
        ...
    }
]
    Parameter Type Description Required
    titleID string Identifier for a Dreamscape title (e.g., alienzoo01) true

    Response Schema

    Field Type Description
    language_code string Two-letter ISO 639-1 code for the language
    language_name string Full name of the language
    written boolean
    audio boolean
    native_name string
